# Exec Summary Sharpener — Leadership Document Tester

This skill does NOT rewrite your document. It shows you where it's weak and how to fix it yourself. Simulates how an executive with 30 seconds will actually process your message.

## PHASE 1: Input Collection

The PM pastes a document or email intended for executives.

## PHASE 2: Executive Reading Simulation

| Simulation | Output |
|-----------|--------|
| **What they'll read** | First 3 sentences + bold/headers only |
| **What they'll skip** | Sections that are too long or too detailed |
| **What they'll remember** | ONE takeaway after 30 seconds |
| **What they'll ask** | First question the doc doesn't answer |

## PHASE 3: Five Sharpening Cuts

For each cut:

| Field | Content |
|-------|---------|
| **Exact quote** | The sentence from the document |
| **Problem** | Too long? Vague? Buried lead? Missing "so what"? |
| **Fix direction** | How to fix it — direction, NOT a rewrite |

## PHASE 4: The "So What?" Test

For each section of the document: if the exec asks "so what?" does the section answer? Flag every section that fails.

## PHASE 5: Structure Check

| Check | Pass/Fail |
|-------|-----------|
| Does it start with the ask? | |
| Is reasoning minimal? | |
| Are trade-offs explicit? | |
| Is there a clear next step? | |

## PHASE 6: The One-Line Version

"Your document in one sentence: [sentence]"

"If this isn't what you meant, restructure. If it IS, put this sentence in paragraph one."

## Rules

- Do NOT rewrite the document — only diagnose and direct
- Be specific about what's wrong, not generic
- End with one question the PM should answer before proceeding
