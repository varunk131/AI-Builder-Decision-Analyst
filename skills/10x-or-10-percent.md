# 10x or 10% — Bet Size Calibrator

This skill answers one question: Is this a 10% improvement or a 10x change? The answer determines whether you're investing correctly.

## PHASE 1: Input Collection

The PM describes a product bet or initiative.

## PHASE 2: Five-Dimension Evaluation

Evaluate the bet against each dimension:

| Dimension | 10% Signal | 10x Signal |
|-----------|-----------|------------|
| **User Behavior Change** | Same thing slightly faster | Something they literally couldn't do before |
| **Market Positioning** | Keeps you competitive | Creates a new category or changes buying criteria |
| **Revenue Ceiling** | Single-digit conversion/retention improvement | New revenue stream, new segment, or new pricing power |
| **Compound Effects** | One-time, linear impact | Network effects, data flywheels, ecosystem effects |
| **Reversibility** | Easy to undo | Permanent strategic commitment |

For each dimension, state where this bet falls and why with specific evidence from the PM's input.

## PHASE 3: Verdict

**This is a [10% / 10x / in between] bet.**

| Diagnosis | Response |
|-----------|----------|
| 10% disguised as 10x | "You're over-investing in a small bet." |
| 10x treated as 10% | "You're under-investing in a transformative opportunity." |
| Correctly calibrated | "Investment matches bet size." |

## PHASE 4: The 10x Reframe

End with: "The 10x version of this idea is: [one sentence]. Is that what you're building?"

## Rules

- Be precise about which dimensions are 10% and which are 10x — most bets are mixed
- Don't inflate or deflate — be honest about the bet size
- End with one question the PM should answer before proceeding
